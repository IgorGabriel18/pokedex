# Pokedex

A website to view and favorite your Pokémons.

### OBJECTIVE

-   Configuration and use of custom imports
-   Improving the handling of API calls and destructuring
-   Creation and use of styling variables with Styled-components
-   Use of ContextApi for global state management


### FEATURES

-   Theme switching
-   Pokémon name search system
-   Filter system by Pokémon type, region, and generation
-   Routing system between pages
-   Load more Pokémon when reaching the end of the page
-   Favorite your Pokémon

### TECHNOLOGIES

-   React.js
-   TypeScript
-   Styled-components
-   ContextApi
-   Axios

### Usage

-   Installation: within the project root, type `pnpm i` in the terminal
-   Initialization: within the project root, type `pnpm dev` in the terminal
